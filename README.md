# Knowledge Zakat (زكاة العلم) 
مشروع "زكاة العلم" هو منصة تعليمية تهدف إلى تعزيز التكافل الفكري والمعرفي، من خلال ربط الخبراء والمتخصصين بالطلاب والباحثين عن المعرفة. يسعى المشروع لتسهيل عملية نقل الخبرات بشكل تطوعي ومنظم.

### مميزات النظام:
* **إدارة المستخدمين:** لوحة تحكم شاملة للأدمن لإدارة الحسابات (حظر، تفعيل، وحذف).
* **نظام النقاشات:** إمكانية طرح الأسئلة والحصول على إجابات من متخصصين.
* تصميم متجاوب وسهل الاستخدام

# التقنيات المستخدمة:
* **Backend:** FastAPI (Python)
* **Frontend:** Svelte 5 & Tailwind CSS
* **Database:** PostgreSQL

---

## Project Description :
**Knowledge Zakat** is an educational platform designed to foster intellectual philanthropy by connecting experts with students. It facilitates the voluntary exchange of knowledge and mentorship in a structured environment.

### Key Features:
* **User Management:** A comprehensive dashboard for admins to manage accounts (Suspend, Activate, and Delete).
* **Discussion System:** Users can post queries and receive expert guidance.
* **Modern UI:** Responsive and user-friendly interface built with cutting-edge technologies.

### Tech Stack:
* **Backend:** FastAPI (Python)
* **Frontend:** Svelte 5 & Tailwind CSS
* **Database:** PostgreSQL

---

## How to Run (كيفية التشغيل)

### Backend:
1. `cd KZ_backend`
2. `pip install -r requirements.txt`
3. `uvicorn main:app --reload`

### Frontend:
1. `cd KZ_frontend`
2. `npm install`
3. `npm run dev`
